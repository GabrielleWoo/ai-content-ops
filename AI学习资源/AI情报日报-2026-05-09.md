## 一、Anthropic / Claude 动态

**2026-05-07 · Claude Opus 4.7 正式发布**
Anthropic 发布新旗舰模型 Claude Opus 4.7，在高难度软件工程任务上显著超越 Opus 4.6，并大幅提升图像分辨率理解能力。
来源：[Introducing Claude Opus 4.7 - Anthropic](https://www.anthropic.com/news/claude-opus-4-7)

**2026-05-06 · Code with Claude 2026 开发者大会：四大核心公告**
1. **算力扩容**：与 SpaceX 签署重磅合作，接管 Colossus 1 数据中心全部算力（300MW、22 万张 NVIDIA GPU，含 H100/H200/GB200），一个月内完成接入，彻底缓解算力告急问题。
2. **Claude Code 额度翻倍**：Pro/Max/团队版调用上限从 5 小时倍增至 10 小时，同时取消高峰时段额度削减限制。
3. **托管智能体三大新能力**：多智能体编排（Multi-Agent Orchestration）、目标结果（Outcomes）、自主推演（Dreaming，研究预览版）。Harvey 使用 Dreaming 后任务完成率提升 6 倍。
4. **Mythos 系列上线**：共推出 18 个 Claude 模型版本，含 Sonnet、Opus 及全新 Mythos 系列，全部向开发者开放。

来源：[钛媒体报道](https://www.tmtpost.com/7978749.html) · [36氪报道](https://36kr.com/p/3798560831822853) · [Simon Willison 现场直播日志](https://simonwillison.net/2026/May/6/code-w-claude-2026/) · [Claude Mythos 预览](https://red.anthropic.com/2026/mythos-preview/)

**2026-05-06 · Claude Design 上线**
Anthropic Labs 推出 Claude Design，支持用户与 Claude 协作创建视觉输出，包括设计稿、原型、幻灯片和一页纸文档。
来源：[Anthropic Release Notes](https://platform.claude.com/docs/zh-CN/release-notes/overview)

**2026-05-05 · Anthropic 深入华尔街：金融智能体 + Microsoft 365 全面整合**
Anthropic 发布面向金融行业的开箱即用 Agent 模板，推出 Excel/PowerPoint/Word/Outlook 四大 Microsoft 365 插件，并接入 Moody's MCP 应用，将 Claude 嵌入金融工作流。

**2026-05-05 · GPT-5.5 Instant 正式成为 ChatGPT 默认模型**

来源：[TechCrunch](https://techcrunch.com/2026/05/05/openai-releases-gpt-5-5-instant-a-new-default-model-for-chatgpt/) · [OpenAI 官方](https://openai.com/index/gpt-5-5-instant/) · [9to5Mac](https://9to5mac.com/2026/05/05/gpt-5-5-instant-makes-chatgpt-more-accurate-while-nixing-gratuitous-emojis/)

**2026-05-07 · OpenAI 向网络安全社区开放 GPT-5.5**
OpenAI 宣布让更多网络安全防御者访问 GPT-5.5，以支持 Cyber 防御用途。
来源：[Axios](https://www.axios.com/2026/05/07/openai-gpt-55-cybersecurity-model)

**2026-04-23 · GPT-5.5 全系发布**
OpenAI 发布 GPT-5.5、GPT-5.5 Thinking、GPT-5.5 Pro 三个版本，定位"最智能最直觉"旗舰模型，擅长 Agentic 编程、Computer Use、知识工作和早期科研。API 定价：$5/1M 输入 tokens，$30/1M 输出 tokens，上下文窗口 1M tokens。
来源：[OpenAI 官方](https://openai.com/index/introducing-gpt-5-5/) · [CNBC](https://www.cnbc.com/2026/04/23/openai-announces-latest-artificial-intelligence-model.html) · [GPT-5.5 System Card](https://openai.com/index/gpt-5-5-system-card/)

来源：[OpenAI 产品发布](https://openai.com/news/product-releases/)
