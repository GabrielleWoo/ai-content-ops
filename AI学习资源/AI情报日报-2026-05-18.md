## 一、Anthropic / Claude 动态

### 2026-05-18 | Claude Opus 4.7 正式发布
Anthropic 推出旗舰模型 Opus 4.7（GA），软件工程与长周期编码任务能力大幅提升，视觉理解支持更高分辨率图像输入。同步上线 **Claude Design**（Anthropic Labs 新产品），可与 Claude 协作生成设计稿、原型、PPT 和单页文档。

### 2026-05-07 | Claude Managed Agents 新增三大能力

## 核心趋势

1. **Anthropic 翻盘** — 职场采用率首次超越 OpenAI，Claude Code 年化收入破 10 亿美元，Claude App 登顶 App Store，"Claude Mania"席卷业界。
2. **AI 即基础设施** — 从 Claude Platform on AWS 到 OpenAI Deployment Company，头部厂商集体向平台/基础设施层转型，企业 AI 部署标准化加速。
3. **Vibe Coding → Agentic Engineering** — Karpathy 推动术语升级，AI 编程范式从"氛围驱动"进化为多 Agent 协同的工程体系，41% 全球代码已由 AI 生成。
4. **语音 AI 进入推理时代** — GPT-Realtime-2 等模型将 GPT-5 级推理引入实时语音，多语言实时翻译进入商用化，人机交互方式将再次重构。
5. **算力军备竞赛升级** — Anthropic 接管 SpaceX Colossus 1（22 万张 GPU），AI 厂商自建或包揽超大规模计算资源成新常态，算力主权竞争白热化。
