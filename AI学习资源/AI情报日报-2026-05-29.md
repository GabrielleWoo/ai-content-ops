## 一、Anthropic / Claude 动态

- **2026-05-28 | Claude Opus 4.8 正式发布**
  Anthropic 发布旗舰新模型 Opus 4.8，在编程、推理、金融分析和知识工作等关键基准上全面超越竞品。默认支持 1M token 上下文窗口（API/Bedrock/Vertex AI），最大输出 128k token。新增「快速模式」，速度提升 2.5x，成本降低 3 倍。同步推出「动态工作流」，支持多个子代理并行运行，以及可调节「响应精力」的控制面板。
  来源：[Axios](https://www.axios.com/2026/05/28/anthropic-opus-release-mythos) · [Bloomberg](https://www.bloomberg.com/news/articles/2026-05-28/anthropic-unveils-new-flagship-ai-model-that-s-better-at-coding) · [TechCrunch](https://techcrunch.com/2026/05/28/anthropic-releases-opus-4-8-with-new-dynamic-workflow-tool/)

- **2026-05-28 | Claude Code 大规模可靠性更新**
  发布宽泛的可靠性和工作流更新，改进后台会话处理、模型与插件管理、更快的 MCP/SDK 启动速度，并修复多项终端崩溃和会话恢复问题。
  来源：[Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **2026-05-25 | Mythos 级模型即将对公众开放**
  Anthropic 宣布已在安全保障方面取得「快速进展」，Mythos 级别模型预计「未来数周内」向所有用户开放。目前 Mythos Preview 因安全评估认为「公开发布不负责任」而暂未上线。
  来源：[The Register](https://www.theregister.com/security/2026/05/25/anthropic-to-release-mythos-class-models-to-the-public/5245596)

- **2026-05-21 | Code with Claude 展示编程未来**
  MIT Technology Review 深度报道 Anthropic 举办的 Code with Claude 活动，探讨 AI 辅助编程的未来形态。
  来源：[MIT Technology Review](https://www.technologyreview.com/2026/05/21/1137735/anthropics-code-with-claude-showed-off-codings-future-whether-you-like-it-or-not/)

- **2026-03（近期）| Claude 用户增长 130%，首超 ChatGPT**
  Claude 从 2026 年 2 月至 3 月扩张 130.1%，美国桌面独立访客达 266 万。曾短暂登顶 Apple App Store，日活用户达 1130 万，付费用户翻倍。开发者群体中约 70% 更倾向于使用 Claude。
  来源：[ppc.land](https://ppc.land/claudes-130-surge-shakes-up-the-ai-chatbot-rankings/)

## 核心趋势

1. **Anthropic 以 Opus 4.8 确立编程领域领先地位**，同时 Mythos 级模型即将公开，AI 能力上限正在快速突破安全评估门槛。

2. **OpenAI 转向实用化深度整合**，个人财务、语音智能、幻觉率大幅降低，显示从「炫技」转向「日常刚需」的产品策略转变。

3. **Vibe Coding 生态系统成熟分化**：Cursor（$2B ARR）、Claude Code、Google AI Studio、Mistral Vibe Agent 各占山头，编程正在对所有人开放。

4. **AI 代理进入「自主长时运行」时代**：OpenClaw、Mistral Vibe、Emergent Wingman 等产品标志着 AI 从「按需响应」转向「无监督后台执行」。

5. **创业门槛断崖式下降**：全栈 AI 工具栈让单人或小团队可以完成以前需要整个工程团队的工作，2026 年创业成本结构正在被重写。
