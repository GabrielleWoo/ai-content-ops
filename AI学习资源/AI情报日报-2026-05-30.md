## 一、Anthropic / Claude 动态

- **2026-05-28｜Claude Opus 4.8 正式发布**
  Anthropic 发布最新旗舰模型 Opus 4.8，在 Opus 4.7 基础上全面提升基准测试表现，诚实性显著增强。新增三大特性：① **动态工作流（Dynamic Workflows）**：Claude Code 单次会话可并行调度数百个子 Agent，支持百万行级代码库迁移；② **思考力度控制（Effort Control）**：用户可调节低/高/额外/最大四档思考深度；③ **快速模式（Fast Mode）**：速度提升 2.5×，成本比前代快速模式低 3×。
  来源：[Anthropic 新闻](https://www.anthropic.com/news) | [Releasebot 发布汇总](https://releasebot.io/updates/anthropic)

- **2026-05-28｜Claude Code 额度翻倍、高峰限制取消**
  claude.ai 用户五小时调用限额翻倍，取消高峰时段的额度削减；新增 Managed Agents Webhooks、多智能体编排、AWS 自托管沙箱。
  来源：[Releasebot Claude Code 更新](https://releasebot.io/updates/anthropic/claude-code) | [蘋果仁报道](https://applealmond.com/posts/312083)

- **2026-05-xx｜Anthropic × SpaceX 算力合作**
  Anthropic 宣布与 SpaceX 达成合作，使用 Colossus 1 数据中心全部算力，新增超过 300 MW 容量与逾 22 万张 NVIDIA GPU，直接回应"算力告急"问题。
  来源：[钛媒体报道](https://www.tmtpost.com/7978749.html)

- **2026-05-21｜Code with Claude 发布会**
  Anthropic 举办 "Code with Claude" 活动，展示了以 Claude Code 为中心的编程未来，引发广泛讨论。
  来源：[MIT Technology Review](https://www.technologyreview.com/2026/05/21/1137735/anthropics-code-with-claude-showed-off-codings-future-whether-you-like-it-or-not/)

- **2026-05-xx｜Claude 跃升 App Store 第 2 名**

## 核心趋势

1. **Claude 全面反超势头显现**：Opus 4.8 发布配合 Code 额度翻倍、SpaceX 算力加持，Anthropic 在开发者侧与消费者侧同步发力，App Store 跻身第 2 名。

2. **OpenAI 以频次应战**：5 月连续推出 GPT-5.5 Instant（降幻觉）、语音模型、理财功能，通过高频迭代守住用户粘性，但话题热度被 Claude 分流明显。

3. **Vibe Coding 进入主流**：Cursor 年化 $2B、Mistral 品牌重塑、Google 全栈工具落地，"自然语言编程"从概念变为产业标配，Collins 词典年度词已锁定。

4. **本地 AI Agent 爆发**：OpenClaw 以 302K Stars 刷新 GitHub 纪录，私有数据、本地运行、多应用整合成用户核心诉求，隐私驱动需求显现。

5. **AI 视频长度突破**：Higgsfield Zephy 近 11 分钟 AI 视频标志生成式视频进入"长内容"时代，视频生成工具竞争即将升温。
