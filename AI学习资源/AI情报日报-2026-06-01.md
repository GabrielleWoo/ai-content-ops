## 一、Anthropic / Claude 动态

**2026-05-28 | Claude Opus 4.8 正式发布**
Anthropic 发布旗舰模型 Opus 4.8，具备更强的代码能力（agentic coding 评分从 64.3% 升至 69.2%）、更低的误导行为率（比 4.7 低约 4 倍），以及对自身代码漏洞的主动标注意识。Claude Code 新增「动态工作流」功能，可拆解超大型任务；Fast Mode 在 Opus 4.8 上降价 3 倍。定价与 Opus 4.7 持平。
- 来源：[TechCrunch](https://techcrunch.com/2026/05/28/anthropic-releases-opus-4-8-with-new-dynamic-workflow-tool/) | [9to5Mac](https://9to5mac.com/2026/05/28/anthropic-upgrades-claude-with-new-opus-4-8-model-heres-whats-new/) | [Axios](https://www.axios.com/2026/05/28/anthropic-opus-release-mythos)

**2026-05-28 | Anthropic 融资 650 亿美元，估值达 9650 亿美元**
Anthropic 宣布完成新一轮融资，估值创历史新高达 9650 亿美元，并承诺将在数周内向所有用户开放 Mythos 级模型。
- 来源：[Fortune](https://fortune.com/2026/05/29/anthropic-raises-65-billion-at-record-965-billion-valuation-promises-mythos-ai-model-in-wide-release-in-coming-weeks-releases-claude-opus-4-8/) | [Bloomberg](https://www.bloomberg.com/news/articles/2026-05-28/anthropic-unveils-new-flagship-ai-model-that-s-better-at-coding)

**2026-04-16 | Claude Opus 4.7 发布**
支持更严谨的长任务处理与更精准的指令跟随，在 Artificial Analysis Intelligence Index 中领跑，agentic 综合能力（GDPval-AA）排名第一。
- 来源：[Releasebot](https://releasebot.io/updates/anthropic)

## 核心趋势

1. **Anthropic 双线布局**：Opus 4.8 填补当前产品空缺，Mythos 储备超强能力等待合规评估完成后全面释放，估值直逼万亿。
2. **OpenAI 以「易用性」反击**：GPT-5.5 系列将幻觉率和操作门槛双降，抢占非技术用户市场。
3. **Vibe Coding 进入主流**：Cursor 年化 20 亿美元、Lovable ARR 2000 万美元、Mistral 品牌押注 Agent，开发者工具格局重塑。
4. **AI Agent 从工具变基础设施**：多 Agent 并行、动态工作流成标配，SaaS 与 Agent 加速融合，传统 SaaS 面临存亡考验。
5. **高端用户集中在 Claude，大众留在 ChatGPT**：月活差距 50 倍，但 Claude 用户质量（高收入、重度专业使用）领先，商业化潜力仍被低估。
