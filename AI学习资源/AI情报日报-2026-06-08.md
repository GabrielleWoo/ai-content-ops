  来源：[Anthropic Newsroom](https://www.anthropic.com/news) | [Claude API Docs](https://platform.claude.com/docs/zh-CN/release-notes/overview)

- **2026-03** — Claude 付费用户数量激增，App Store 排名从 #131 一跃升至 #1，日活突破 1130 万，免费注册增长 60%，付费用户翻倍以上。

## 核心趋势

1. **Anthropic 进入"双轨"策略**：公开版（Opus 4.8）全面铺开，内部高阶版（Mythos）严格管控，拉开与竞争对手的能力差。
2. **agentic coding 成主战场**：Claude Code Dynamic Workflows、Mistral Vibe、GPT-5.3-Codex、Cursor 均在 Agent 化编程方向发力，从"辅助写代码"演进为"自主完成工程任务"。
3. **OpenAI 模型迭代提速 + 退役加速**：GPT-5.5 系列接连登场，老模型（4.5、o3）密集退役，版本跨越步伐明显加快。
4. **开源 + 低门槛工具爆发**：OpenClaw 爆火、Microsoft 自研模型、Google/Kaggle 免费课程，显示 AI 能力正快速向平民化、无代码化方向下沉。
5. **Vibe Coding 从概念走向产业**：Collins 年度词、Cursor $20 亿 ARR、Mistral 品牌重塑，标志 Vibe Coding 已超越炒作期，进入规模化落地阶段。
