## 一、Anthropic / Claude 动态

- **2026-06-09** Anthropic 企业市场份额在 4 月首次超越 OpenAI（34.4% vs 32.3%），Claude 单月网页访问量增长 **306%**（1 月 2.03 亿 → 4 月 8.24 亿）。[来源](https://www.techtimes.com/articles/318024/20260608/chatgpt-holds-547-ai-chatbot-traffic-claude-306-why-that-number-misleads.htm)

- **2026-06-02** Anthropic 将 **Project Glasswing**（Mythos 测试计划）扩展至全球 15 个以上国家的 150 家合作伙伴，Mythos 公开发布预计"数周内"到来。[来源](https://www.cnbc.com/2026/06/02/anthropic-mythos-ai-project-glasswing.html)

- **2026-06-01** Anthropic 向 SEC **秘密提交 IPO 申请**，预计估值约 1 万亿美元，5 月年化营收（ARR）已达 **470 亿美元**。[来源](https://www.openpr.com/news/4541471/anthropic-files-confidentially-for-a-2026-ipo-as-annual-run-rate)

- **2026-05-29** Anthropic 宣布完成 **650 亿美元**融资，估值达 **9650 亿美元**，成为全球最高估值 AI 初创公司。[来源](https://fortune.com/2026/05/29/anthropic-raises-65-billion-at-record-965-billion-valuation-promises-mythos-ai-model-in-wide-release-in-coming-weeks-releases-claude-opus-4-8/)

- **2026-05-28** 发布 **Claude Opus 4.8**，新增"动态工作流（Dynamic Workflow）"工具，支持跨代码库端到端并行任务、迁移和复杂工程工作，内置验证与进度保存，覆盖 CLI / Desktop / VS Code / API / 主流云平台。[来源](https://techcrunch.com/2026/05/28/anthropic-releases-opus-4-8-with-new-dynamic-workflow-tool/)

- **2026-04-07** **Claude Mythos Preview** 内部曝光——Anthropic 称其为"迄今能力最强的前沿模型"，在代码与漏洞扫描上实现"破坏性"跨代提升，目前仅向少数美国机构开放，官方认为"公开发布不负责任"。[来源](https://zhuanlan.zhihu.com/p/2025121628577625231)

- **Claude Code** 新增管理员权限自定义角色（Enterprise），让成员可在不获得 Owner 权限的情况下访问计费和隐私设置。[来源](https://releasebot.io/updates/anthropic/claude)

## 二、OpenAI 动态

- **2026-06-09** **GPT-5.5 Instant** 成为 ChatGPT 新默认模型，取代 GPT-5.3 Instant，带来更精准的答案、更强的图像与 STEM 辅助，以及更丰富的 Plus/Pro 上下文。[来源](https://releasebot.io/updates/openai/chatgpt)

- **2026-06 退役预告** GPT-4.5 将于 **6 月 27 日**从 ChatGPT 下线；o3 将于 **8 月 26 日**退役（90 天日落期）。[来源](https://help.openai.com/en/articles/9624314-model-release-notes)

- **2026-06（近期）** 发布 **GPT-5.3-Codex**，最强 agentic 编程模型，结合 Codex + GPT-5 训练栈，速度提升约 25%，定位从代码生成升级为通用编程 Agent。[来源](https://openai.com/index/introducing-gpt-5-5/)

- **2026-04-24** **GPT-5.5 / GPT-5.5 Pro** 正式上线 API，擅长代码调试、在线研究、数据分析、文档与电子表格创建等任务。[来源](https://openai.com/index/introducing-gpt-5-5/)

- **2026-04-14** **GPT-6**（代号"Spud/土豆"）发布，以记忆与个性化为核心，上下文窗口达 **200 万 Token**。[来源](https://cloud.tencent.com/developer/article/2659606)

- **2026-04-14** 发布网络安全专用模型 **GPT-5.4-Cyber**，针对防御性安全工作微调。[来源](https://companies.caixin.com/m/2026-04-15/102434111.html)

- **2026-06-02** **Microsoft** 在 Build 大会发布 **MAI-Code-1-Flash**，可将文字描述直接转换为应用/网站源代码，旨在降低对 OpenAI 的依赖。[来源](https://www.cnbc.com/2026/06/02/microsoft-unveils-new-ai-models-lessen-reliance-on-openai-lower-costs.html)

## 三、AI Builder / 工具

- **Vibe by Mistral**：Le Chat 品牌升级为 **Vibe**，单一 Agent + 单一许可证覆盖办公与代码场景，Work Mode 支持网页/移动端，流式展示长任务进度并自动选择工具完成复杂工作。[来源](https://mistral.ai/news/vibe-agent/)

- **Google Antigravity 2.0**：新一代 Agent 优先开发平台，发布独立桌面应用，支持多 Agent 并行编排，任何人都能成为 Builder。同期推出全栈 Vibe Coding 体验（Google AI Studio），可将 Prompt 直接转化为生产级应用。[来源](https://blog.google/innovation-and-ai/technology/developers-tools/full-stack-vibe-coding-google-ai-studio/)

- **Google I/O 2026** 发布 **Gemini 3.5 Flash**（定价 $1.50/$9.00），首批在 AI Studio 部署的 2 个应用免费，无需信用卡。共发布逾 100 项 AI 公告。[来源](https://blog.google/innovation-and-ai/technology/ai/google-io-2026-all-our-announcements/)

- **Dify**：可视化 LLM 应用构建器，拖拽式界面支持 RAG Pipeline、聊天机器人与 AI Agent，无需编写 YAML 配置，成为 6 月 Builder 圈高频推荐工具。[来源](https://launchtoolsai.com/best/emerging-ai-developer-tools-june-2026/)

- **Cursor**：专业开发者 Vibe Coding 首选工具，年化收入已达 **20 亿美元**（2026 年初）。[来源](https://manus.im/blog/best-vibe-coding-tools)

- **Emergent**（YC 支持）：多专用 AI Agent 协作平台，自动化设计、编码、部署全栈 Web/移动应用，产出接近生产就绪。[来源](https://wavespeed.ai/blog/posts/june-2026-ai-launch-wave/)

- **Google + Kaggle AI Agents Vibe Coding 课程**（6 月）：免费公开课，聚焦 Agent 开发与 Vibe Coding 实践。[来源](https://blog.google/innovation-and-ai/technology/developers-tools/kaggle-genai-intensive-course-vibe-coding-june-2026/)
