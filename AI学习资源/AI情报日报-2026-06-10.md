## 一、Anthropic / Claude 动态

**2026-06-10 | Claude Fable 5 正式发布 — Mythos 级模型面向公众开放**
Anthropic 于 6 月 9 日发布 Claude Fable 5，这是 Mythos 系列中首个对外公开的模型。定价为输入 $10/百万 Token、输出 $50/百万 Token，支持 100 万 Token 上下文窗口及最高 128K 输出 Token。内置安全分类器，若触发高风险请求（如网络安全、生物领域滥用）将自动路由至 Opus 4.8，触发率约 <5%。可通过 Claude API、Amazon Bedrock、Google Vertex AI、Microsoft Foundry 调用。
- [TechCrunch 报道](https://techcrunch.com/2026/06/09/anthropic-released-claude-fable-5-its-most-powerful-model-publicly-days-after-warning-ai-is-getting-too-dangerous/)
- [CNBC 报道](https://www.cnbc.com/2026/06/09/anthropic-mythos-claude-fable-5.html)
- [Cryptopolitan](https://www.cryptopolitan.com/anthropic-launches-claude-fable-5/)

**2026-06-09 | Claude 接入 Apple Foundation Models 框架**
Anthropic 发布 Swift 包，让苹果开发者可在 iOS 27 / macOS 27 / iPadOS 27 / visionOS 27 / watchOS 27 上通过 Apple Foundation Models 框架直接调用 Claude 处理复杂工作流。
- [Releasebot Anthropic 更新页](https://releasebot.io/updates/anthropic)

**2026-06-02 | Anthropic 将 Mythos 访问权扩展至 150+ 机构**
通过 Project Glasswing 计划，Mythos 模型已向全球 15 个以上国家的 150 个组织开放，聚焦网络防御方向。
- [CNBC 报道](https://www.cnbc.com/2026/06/02/anthropic-mythos-ai-project-glasswing.html)

**2026-05-28 | Claude Opus 4.8 发布，新增动态工作流工具**
Opus 4.8 在编码和知识型工作上有显著提升，定价与前版相同，新增"dynamic workflow"工具支持复杂 Agent 流程编排。
- [TechCrunch 报道](https://techcrunch.com/2026/05/28/anthropic-releases-opus-4-8-with-new-dynamic-workflow-tool/)
- [Axios 报道](https://www.axios.com/2026/05/28/anthropic-opus-release-mythos)

**2026-06（持续更新）| Claude Code 新功能：Safe Mode、/cd 命令**
Claude Code 新增 safe mode、/cd 命令、隐藏内置技能设置，优化远程 session、后台 Agent 和企业策略执行；同时修复 Windows/macOS UI 问题并降低 CPU 占用。

## 核心趋势

1. **Mythos 级模型公开化**：Anthropic Fable 5 将此前受限的最强模型推向市场，开创"安全路由"定价新范式，预计 OpenAI 将加快对应级别模型发布节奏。
2. **Agent 化是主旋律**：Claude Code、Cursor、Mistral Vibe、Google AI Studio 均在向"从需求到部署"全流程 Agent 演进，纯对话模式加速退场。
3. **Vibe Coding 完成主流化**：从 2025 年新词到 2026 年年度词汇，AI 编码工具年化营收（Cursor $2B+）证明这不是概念，而是真实市场。
4. **苹果生态进场**：Apple Foundation Models 框架打通 Claude，意味着设备端与云端 AI 融合正式提速，iOS/macOS 开发者将进入新工具周期。
5. **开源社区爆发**：OpenClaw（210K Stars）和 CLAUDE.md（109K Stars）表明社区力量正在定义 AI 工具的使用规范，远超官方文档的影响力。
