## 一、Anthropic / Claude 动态

- **2026-06-09** | Anthropic 正式发布 **Claude Fable 5**，这是首个面向公众开放的 Mythos 级模型，覆盖 Claude API、AWS Bedrock、Vertex AI 和 Microsoft Foundry。Fable 5 在软件工程、知识工作和视觉理解方面超越此前所有公开模型，但在网络安全、生物、化学等高风险领域设有硬限制，自动回退至 Claude Opus 4.8。Pro/Max/Team/Enterprise 用户可免费使用至 6 月 22 日。

## 核心趋势

1. **Mythos 时代开启** — Anthropic Fable 5 公开发布标志着 AI 能力进入新量级，但安全硬限制成为常态，模型"双轨"（公开版 vs 受信机构版）策略或成行业新范式。

2. **OpenAI 高频迭代** — GPT-5.3-Codex / 5.4 / 5.5 Instant 密集上线，版本命名碎片化，市场竞争进入月月更新节奏；同时主动清退老模型（o3、GPT-4.5）以简化产品线。

3. **Vibe Coding 主流化** — 年化规模约 47 亿美元，开发者采用率接近 90%，Cursor、Mistral Vibe、Google AI Studio 全栈方案三足鼎立，Claude Code 终端生态持续扩张。

4. **Claude 生态份额暴涨** — Claude 流量份额一年内从 1.6% 升至 8.9%，职场用户粘性超越所有主流 AI 平台，Claude Code 在开发者社区病毒式传播推动品牌快速渗透。

5. **AI Agent 企业落地提速** — 预计年底 40% 企业应用内置 AI Agent（2025 年仅 5%），"从聊天到行动"是 2026 下半年最核心的产品形态转变。
