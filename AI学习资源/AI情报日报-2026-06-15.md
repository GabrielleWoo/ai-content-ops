## 一、Anthropic / Claude 动态

- **2026-06-15** — Anthropic 今日正式调整计费方式，将互动式使用与程式化（API/自动化）使用切割为两个独立额度池，不再互相占用，防止低价订阅"吃掉"高用量配额。
  来源：[Claude 订阅方案大更新 — 未来商务](https://fc.bnext.com.tw/articles/view/4622)

- **2026-06-09** — Anthropic 发布 **Claude Fable 5**，为 Mythos 系列首个公开可用版本，同步推出仅向受信任机构开放的 **Claude Mythos 5**。Fable 5 在软件工程、知识工作与视觉任务上表现突出，但在网络安全、生物、化学等高风险领域启用硬性安全限制并回退至 Opus 4.8。Pro/Max/Team/企业版用户免费使用期至 6 月 22 日，6 月 23 日起需消耗积分。
  来源：[TechCrunch — Anthropic releases Claude Fable 5](https://techcrunch.com/2026/06/09/anthropics-claude-fable-5-is-a-version-of-mythos-the-public-can-access-today/) | [NBC News](https://www.nbcnews.com/tech/security/fable-5-anthropic-release-public-mythos-claude-model-rcna349104) | [新浪科技](https://finance.sina.com.cn/tech/roll/2026-06-10/doc-iniawttz1374324.shtml)

- **2026-06-（近期）** — Claude Code 更新：新增嵌套子 Agent、更智能的模型与区域选择、新插件搜索、Chrome / VSCode / 终端工作流改进，以及大量 Session、Agent、权限与 UI Bug 修复。
  来源：[Claude Code Updates — Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **2026-06-（近期）** — Anthropic 官方 X 账号公布：Claude 在开放性编程问题上的成功率已达 **76%**，较 6 个月前跃升 50 个百分点，众多工程师认为 Claude 代码质量已与人类持平。

## 核心趋势

1. **Mythos/Fable 时代开启**：Anthropic 以 Fable 5 将顶级能力带入公开市场，但以硬限制换取安全背书，监管博弈正在塑造模型能力边界。
2. **OpenAI 迭代提速，GPT-5.x 系列密集发布**：从 5.5 到预期 5.6，平均迭代窗口压缩至 6 周，Codex 的全面开放标志着 Agentic 编码进入主流。
3. **Claude 的技术心智份额持续扩大**：连竞争对手的工程师都在使用 Claude，76% 的开放编程成功率使其在代码质量上逼近人类基线。
4. **Vibe Coding 生态成熟**：Cursor 年化 $20 亿美元、Mistral Vibe 全面转型、Google AI Studio 支持全栈开发，vibe coding 已从概念进入行业基础设施阶段。
5. **计费与访问权重组**：Anthropic 今日起拆分互动/程式化用量，预示高能耗 AI 应用的成本结构将进一步透明化，影响企业 AI 预算规划。
