## 一、Anthropic / Claude 动态

- **2026-06-19** — **企业级 MCP Connector 上线**：Claude 新增企业托管 MCP 连接器访问能力，首批支持 Okta，管理员统一配置后，用户首次登录即可零配置获得连接权限。
  来源：[Anthropic Release Notes – June 2026](https://releasebot.io/updates/anthropic)

- **2026-06-12** — **美国出口管制令下线 Fable 5 / Mythos 5**：美国政府以出口管制指令要求 Anthropic 暂停 Fable 5 与 Mythos 5 的全球访问，两者目前均已下线，原因待官方进一步说明。
  来源：[仅仅间隔11天，Anthropic发布Claude Fable 5 – 证券时报](https://www.stcn.com/article/detail/3952926.html)

- **2026-06-09** — **Claude Fable 5 正式发布**：Anthropic 发布首个面向公众开放的 Mythos 级模型 Claude Fable 5，同步推出限制访问的 Claude Mythos 5。Mythos 5 已在蛋白质设计等药物研发任务中达到专业研究人员水平，并被称为"全球最强网络安全能力"模型。同日在 Claude API、AWS、Amazon Bedrock、Vertex AI、Microsoft Foundry 多平台上线。
  来源：[Anthropic releases Claude Fable 5 – TechCrunch](https://techcrunch.com/2026/06/09/anthropics-claude-fable-5-is-a-version-of-mythos-the-public-can-access-today/)

- **2026-06-初** — **Claude Managed Agents 公测**：Claude 推出托管 Agent 公开测试版，支持定时（cron）调度、CLI 工具调用及 vault 存储环境变量，实现无人值守自动化流程。
  来源：[Anthropic Release Notes – June 2026](https://releasebot.io/updates/anthropic)

- **2026-06-初** — **Claude Code 持续迭代**：新增语言感知会话标题、更丰富的页脚链接徽章、更智能的 Bedrock 凭证缓存，并改善 agent 会话启动体验。
  来源：[Claude Code Updates – June 2026](https://releasebot.io/updates/anthropic/claude-code)

- **2026-06-初** — **Claude Design 更新**：新增直接画布编辑、更强布局控制，与 Claude Code 同步更紧密，让设计保持跨项目对齐。
  来源：[Claude Updates – June 2026](https://releasebot.io/updates/anthropic/claude)

- **增长数据** — Claude 网页访问量 1 月至 4 月增长 306%（从 2.03 亿到 8.24 亿），美国移动端 DAU 年初至今增长 1,100%，人均使用时长已超 ChatGPT。
  来源：[Gen AI Chatbots: June 2026 – Apptopia](https://apptopia.com/en/insights/gen-ai-chatbots-june-2026/)
